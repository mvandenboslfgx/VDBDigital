# Vertalingen (i18n)

**Standaardtaal:** Nederlands (zakelijk). **Fallback:** Engels.

Alleen **gebruikersinterface-tekst** wordt vertaald. Databasevelden, API-responses en logs worden niet vertaald.

## Structuur

- **locales/nl/common.json** — Nederlandse teksten (default)
- **locales/en/common.json** — Engelse teksten (fallback)
- **lib/i18n.ts** — `getTranslations(locale)` voor server; fallback op nl bij ontbrekende key
- **components/I18nProvider.tsx** — Client-provider + `useTranslations()` hook

## Gebruik

### Server components (pagina’s, layouts)

```ts
import { getTranslations } from "@/lib/i18n";

export default async function Page() {
  const t = getTranslations("nl");
  return <h1>{t("dashboard.title")}</h1>;
}
```

### Client components

```tsx
"use client";
import { useTranslations } from "@/components/I18nProvider";

export default function MyComponent() {
  const { t } = useTranslations();
  return <button>{t("errors.retry")}</button>;
}
```

## Sleutelnamen (voorbeelden)

- **dashboard.*** — Dashboard, overzicht, gebruik, rapporten, instellingen, enz.
- **billing.*** — Facturatie, huidig plan, abonnement beheren
- **settings.*** — Instellingen, e-mail, plan, portal
- **notifications.*** — Meldingen
- **errors.*** — Fouttitels en -berichten
- **auth.*** — Inloggen, labels, foutmeldingen
- **ai.*** — AI-tools, snelle tools, upgrade-teksten
- **calculators.*** — Titel en ondertitel bedrijfscalculators
- **admin.*** — Gebruikersbeheer, systeemstatus, metrics, enz.
- **nav.*** — Navigatie (bijv. Scans)
- **common.*** — Openen, Terug naar site

## Terminologie (NL)

Consistent gebruik: **Dashboard**, **Rapportages**, **Instellingen**, **Facturatie**, **Abonnement**, **Gebruik**, **AI-tools**, **Bedrijfscalculators**, **Gebruikersbeheer**, **Systeemstatus**.

## Root layout

De app is gewrapped in `<I18nProvider locale="nl">` in **app/layout.tsx**. Voor meertaligheid kan `locale` later uit cookie of URL komen.
