# Translation files

UI translations are loaded from `locales/{locale}/common.json` (nl, en, de).

- **Dutch (nl)** – default language, professional Dutch for the entire platform.
- **English (en)** – full UI translation.
- **German (de)** – full UI translation.

The active locale is stored in the `NEXT_LOCALE` cookie and can be changed via the language switcher in the navigation (Nederlands | English | Deutsch).
